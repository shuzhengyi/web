## 需要实现的功能

### 1. 模板记忆学习功能

**需求**：手动调整映射后自动保存映射规则（基于表头列名指纹），下次上传相同结构模板时自动识别并应用。

**实现方案**：
- 创建 `template-memory.ts` 工具文件，存储映射规则
- 使用 localStorage 或后端数据库保存映射规则
- 生成表头指纹（所有列名的哈希值）
- 导入时比对指纹，自动应用保存的映射规则

### 2. 完善手动映射功能

**需求**：自动识别失败时，用户可手动选择Excel列与系统字段的对应关系。

**实现方案**：
- 完善字段映射弹窗组件
- 添加拖拽或下拉选择方式
- 支持清空映射和重置操作

### 3. 大文件处理进度反馈

**需求**：1000条数据导入不卡顿，无内存溢出。

**实现方案**：
- 添加实时进度条显示
- 使用 Web Worker 处理大数据量
- 添加分批导入机制

## 风险评估

| 风险 | 影响 | 应对措施 |
|------|------|---------|
| localStorage 容量限制 | 存储大量映射规则可能超出限制 | 定期清理旧规则，保留最近N条 |
| 指纹冲突 | 不同模板生成相同指纹 | 使用更复杂的哈希算法 |
| 性能问题 | 大量数据处理耗时 | 使用 Web Worker 异步处理 |
